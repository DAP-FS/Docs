Example Structure
-----------------

Each example follows this format:

1. **Problem Statement:** Clear description of the task
2. **Dataset:** Information about the data used
3. **Implementation:** Complete Python code with explanations
4. **Visualization:** Plots and charts for interpretation
5. **Analysis:** Insights and observations
6. **Try It Yourself:** Exercises and modifications
